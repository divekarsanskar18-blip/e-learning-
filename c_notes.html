PROGRAMMING IN C

Unit - I: Basics of C Programming

  • Introduction to C language – History, Features, Applications
  • Structure of C program, Compilation and Execution process
  • Character set, Tokens, Keywords, Identifiers
  • Constants, Variables, Data types
  • Input and Output functions (printf, scanf)
  • Operators – Arithmetic, Relational, Logical, Assignment, Increment/Decrement

Unit - II: Control Structures

  • Decision making statements: if, if-else, nested if
  • Switch-case statement
  • Looping statements: while, do-while, for
  • Break, continue and goto statements
  • Examples and programs based on control structures

Unit - III: Arrays & Strings

  • Definition and declaration of arrays
  • One-dimensional and multi-dimensional arrays
  • String handling in C
  • String functions: strlen, strcpy, strcmp, strcat
  • Applications of arrays and strings

Unit - IV: Functions & Storage Classes

  • Need of functions, advantages
  • Types of functions: library and user-defined
  • Function declaration, definition and calling
  • Categories of functions (with/without arguments, with/without return value)
  • Recursion in C
  • Storage classes: auto, static, extern, register

Unit - V: Pointers, Structures & File Handling

  • Pointers – Definition, declaration, advantages
  • Pointers and arrays, pointers and functions
  • Structures and unions – definition, declaration, accessing members
  • Nested structures and arrays of structures
  • Introduction to file handling in C – fopen, fclose, fprintf, fscanf
