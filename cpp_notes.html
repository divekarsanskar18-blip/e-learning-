1.1 Procedure Oriented Programming (POP) verses Object Oriented Programming (OOP)

Procedure Oriented Programming (POP)

Definition: Programming paradigm based on functions (procedures) that operate on data. Focus: Functions/procedures are the main building blocks. Data is secondary. Examples: C, Fortran, Pascal.

Characteristics:

Programs are divided into functions.
1. Data is often global and shared among functions.
2. Functions operate on data directly.
3. Top-down approach in program design.
4. Less secure since global data can be accessed/modified by any function.
5. Difficult to manage when program grows large.

Object Oriented Programming (OOP)

Definition: Programming paradigm based on objects that encapsulate both data and behavior. Focus: Objects and classes. Data and functions are bundled together. Examples: Java, C++, Python, C#, Ruby.

Characteristics:

Programs are divided into classes and objects.
1.Data is hidden and accessed only through methods (Encapsulation).
2.Supports Inheritance (code reusability).
3.Supports Polymorphism (same function name can work in different ways).
4.Bottom-up approach in program design.
5.More secure since data is accessed through defined interfaces.

1.Abstract Data types

In C++, complex data types called Abstract Data Types (ADT) can be created using classes. Programmers can construct abstract data types through classes and structures in C++ to combine data with their associated functions within a single entity. It allows programmers to produce user-made data types that behave similarly to native program types while extending their capabilities.

2.Machine Independent or Portable

C++ is a portable language and programs made in it can be run on different machines. The functionality of C++ extends from high-level elements, such as abstraction and OOP as well as low-level capabilities, including direct memory access and pointers, which makes it a mid-level language.

3.Mid-level programming language

C++ includes both low-level programming and high-level language, so it is known as a mid-level and intermediate programming language. it is used to develop system applications such as kernel, driver, etc. In its middle-level status, C++ unites functionality from both high-level (abstraction and Object Oriented Programming along with standard libraries) and low-level (direct memory access and pointer functionality) language elements.

4.Structured programming language

C++ is a structured programming language. In C++, we can divide the program into several parts using functions. The modular design structure makes it easier to create code that is easy to maintain and reuse. The addition of OOP features in C++ maintains the function-based programming structure from its parent language, C.
